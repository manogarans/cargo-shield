# Cargo Security IoT System 🚚

## Overview
This project detects cargo theft using IoT sensors and real-time monitoring.

## Features
- GPS tracking
- Door status detection
- Shock detection
- Firebase real-time dashboard
- GSM communication

## Hardware
- ESP32
- GPS Module
- SIM800L
- MPU6050
- Reed Switch
- Buzzer

## Software
- Arduino IDE
- Firebase Realtime Database
- Web Dashboard (HTML + JS)

## Architecture
Sensors → ESP32 → GSM → Firebase → Web Dashboard
